Portfolio

Jinterest

A Pinterest-like Javascript application that generates images using Flickr's photo search API and allows users to tag and save photos via Parse SDK. Users visting the same page can chat with each other using the cloud-hosted real-time messaging system PubNub.

Barterville

Don't have any money, but need some new stuff? Enter Barterville. Built on Rails, Barterville allows you to trade your goods with other users goods in this virtual marketplace. Extensive use of AJAX for a seamless user-experience.

Learnwithme

Providers like Khan Academy, Yale, and Stanford are putting up thousands of lectures for free on the internet. But how do you find, say, the best intro accounting course? Learnwithme solves that problem by aggregating thousands of courses from across the internet (more everyday through cron jobs!), and letting you take and share notes on them with fellow learners.
